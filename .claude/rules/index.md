## 关于记忆系统

这是 Wolf Pack 插件的多层级记忆系统，用于记录项目决策、模式和问题。

### 记忆类别

- **决策 (decisions/)**: 记录重要技术决策及其理由
- **模式 (patterns/)**: 记录可复用的成功模式
- **问题 (issues/)**: 记录遇到的问题和解决方案
- **上下文 (context/)**: 项目上下文信息
- **归档 (archived/)**: 已归档的历史记忆

### 使用方法

```bash
# 记录新记忆
/wolf-memory save issue [描述]
/wolf-memory save decision [描述]
/wolf-memory save pattern [描述]

# 搜索记忆
/wolf-memory search [keyword]

# 加载记忆
/wolf-memory load [id]
```
